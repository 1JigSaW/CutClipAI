# Настройка нового Chrome профиля с верифицированным аккаунтом

## Шаги:

### 1. Запустить VNC на сервере:
```bash
cd ~/CutClipAI
docker-compose -f docker-compose.production.yml exec -d worker bash /app/start_vnc.sh
```

### 2. На твоем Mac создать SSH туннель:
```bash
ssh -L 5900:localhost:5900 root@45.135.234.33
```

### 3. Подключиться к VNC:
- Открой VNC Viewer (или любой другой VNC клиент)
- Подключись к: `localhost:5900`
- Должен открыться виртуальный рабочий стол с Chrome

### 4. Создать новый профиль в Chrome:
1. В Chrome открой: `chrome://settings/people`
2. Нажми кнопку **"Добавить"** (Add person) или **"Добавить профиль"**
3. Создай новый профиль (например, "Profile 1" или "YouTube Verified")
4. В новом окне Chrome открой `youtube.com`
5. **ВАЖНО:** Залогинься в аккаунт, который **УЖЕ ВЕРИФИЦИРОВАН по возрасту** (18+)
6. Открой любое возрастное видео для проверки (например: https://www.youtube.com/watch?v=0KE1ayhUYGY)
7. Убедись, что видео открывается без запроса верификации
8. Закрой Chrome

### 5. Проверить новый профиль:
```bash
cd ~/CutClipAI
./check_chrome_profiles.sh
```

Должен появиться новый профиль (например, "Profile 1") и он должен работать с возрастными видео.

### 6. Готово!
Код автоматически переберет все профили и найдет рабочий.

## Важно:
- Chrome должен запускаться с флагом `--password-store=basic` (уже настроено в `start_vnc.sh`)
- Аккаунт должен быть **УЖЕ ВЕРИФИЦИРОВАН** по возрасту на YouTube
- Не нужно верифицировать аккаунт через VNC - он должен быть верифицирован заранее
